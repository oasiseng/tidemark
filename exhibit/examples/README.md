# Example Exhibits

## Why Examples Aren't Included

Completed Formwork Elevation Reference Exhibits are PE-stamped client deliverables and contain project-specific information (addresses, elevations, property details). They are not included in this open-source repository.

## What to Expect

A completed FERE typically looks like this:

**Sheet A.1 — Site Plan / Formwork Elevation Exhibit**
- Site plan with property boundaries and proposed building
- Benchmark photo exhibits (2–3 annotated field photos)
- FEMA FIRM flood map exhibit
- Foundation detail showing FFE relationship to grade
- Elevation control notes with benchmark table
- General notes and disclaimers
- GIS/contour exhibit (if available)
- PE stamp, signature, and date

**Sheet count:** Typically 1–2 sheets depending on complexity.
